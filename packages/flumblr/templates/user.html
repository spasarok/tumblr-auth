TumbleAuth

An example OAuth application using Flask.

Congrats! Now you can do stuf with the Tumblr API!

Post to Your Blog

Enter your blog name, post title, and post body to post to your blog. This will only work if you have authenticated.




Get Blog Data

Enter your blog name to gets its data in JSON format. This will only work if you have authenticated (although nothing inherent to the API function actually requires authentication).
